# Realtime Chat System

A modern full-stack real-time chat application that enables users to communicate instantly through a fast, secure, and responsive interface.

## 🚀 Features

* 🔐 User Authentication (Sign Up & Login)
* 💬 Real-time messaging
* 👤 User profiles
* 🟢 Online/Offline user status
* 📱 Responsive UI for desktop and mobile
* 🎨 Modern and intuitive interface
* 🔒 Secure authentication and protected routes

## 🛠️ Tech Stack

### Frontend

* React.js
* Vite
* Tailwind CSS
* Zustand
* Axios
* React Router

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* Socket.IO
* JSON Web Token (JWT)
* bcrypt

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/Ravan-dtr/Realtime-chat-system.git
```

### 2. Navigate to the project

```bash
cd Realtime-chat-system
```

### 3. Install dependencies

```bash
npm install
```

Install frontend dependencies:

```bash
cd frontend
npm install
```

Install backend dependencies:

```bash
cd ../backend
npm install
```

## ▶️ Running the Application

### Start the backend

```bash
cd backend
npm run dev
```

### Start the frontend

```bash
cd frontend
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```
